  • Welcome to my friendly calculator

  • The calculator can add, subtract, multiply and divide.
  • When the user hits the “=” button or the "enter" button, the result will appear in the result window. If a user would like to save the result before future updates, he/she needs to press "enter" on the keyboard.
  • The calculator respects the order of operations: * and / after will be calculated before + and -
  • To erase a result in the input only, a user has to hit C. To erase a result, a user has to hit Delete. A user can delete any numbers in input, but only before he/she pressed enter. After a user hits enter, the result can be deleted by hitting the Delete button.
  • The user can chain operations either by multiple operations in the window (1 + 2 * 4 / 3)
